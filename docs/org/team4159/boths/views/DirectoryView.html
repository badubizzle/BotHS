Class DirectoryView

  • java.lang.Object
    • org.team4159.boths.View
      • org.team4159.boths.views.DirectoryView

  • public class DirectoryView
extends View
    This view serves files from a resource directory.

    If you have a folder in your resources folder named "www" and have a file in it named "abc.html", you can define a Route with "/files/" as the path prefix; that file can now be accessed at http://address/files/abc.html.
